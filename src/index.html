vizType

1. choose a data file from your computer

must be CSV format

with criteria headers as first row, eg:

        
        city,state,population,land area
        seattle,WA,652405,83.9
        new york,NY,8405837,302.6
        boston,MA,645966,48.3
        kansas city,MO,467007,315.0

2. Choose rankable criteria

which criteria do you wish to include in rankings?

- ie. don't pick non-numeric fields

- nor any non-rankable identifiers

3. Minimise or Maximise

default is that low numbers are better

- select fields where high values are good

4. Choose an ID field

- used to identify candidates

- eg. name or ID field

5. Finally, choose a visualisation type

- include little graphics

- and describe/explain them?
